# SESSION 9 – NVIDIA (Image Processing)

## Aim
To convert an RGB image pixel into grayscale using the luminance formula.

## Dataset
CIFAR-10 Dataset
https://www.cs.toronto.edu/~kriz/cifar.html

## Objective
To compute grayscale intensity from RGB values for image preprocessing.

## Formula
Gray = 0.3R + 0.59G + 0.11B

## Input
R = 120
G = 150
B = 200

## Procedure
Step 1: Read RGB values
Step 2: Apply grayscale formula
Step 3: Compute intensity
Step 4: Display result

## Algorithm
Step 1: Start program
Step 2: Input R, G, B values
Step 3: Apply formula:
        Gray = 0.3R + 0.59G + 0.11B
Step 4: Compute grayscale value
Step 5: Display result
Step 6: Stop program

## Code Logic
gray = 0.3*R + 0.59*G + 0.11*B

## Output
Gray = 151

## Result
Grayscale value computed successfully.

## Industry Application
Used in NVIDIA Vision AI systems for preprocessing images.
